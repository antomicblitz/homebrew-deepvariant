# Antomicblitz Deepvariant

## How do I install these formulae?

`brew install antomicblitz/deepvariant/<formula>`

Or `brew tap antomicblitz/deepvariant` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "antomicblitz/deepvariant"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
